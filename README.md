# Android Auth Flow 📱

Projeto desenvolvido para a disciplina de **Desenvolvimento de Sistemas** (**IF Sudeste MG**). Este repositório marca o meu primeiro contato com o **Android Studio**, focando na construção de um fluxo completo de autenticação e gerenciamento de estados entre telas.

## 🎯 Objetivo
Implementar as bases de um sistema de acesso ao usuário, garantindo a integridade dos dados inseridos e a navegação fluida entre os módulos de login, cadastro e visualização de perfil.

## 🚀 Funcionalidades
* **Tela de Login:** Validação de credenciais específicas e tratamento de campos vazios.
* **Tela de Cadastro:** Formulário estruturado com 12 campos (pessoais e de endereço).
* **Fluxo de Dados:** Captura e exibição dinâmica do e-mail do usuário logado na tela principal.
* **Navegação:** Gerenciamento de pilhas de *Activities* e botões de retorno/logout.

## 🛠️ Tecnologias e Ferramentas
* **Linguagem:** Java
* **Interface:** XML (Layouts estruturados)
* **IDE:** Android Studio
* **SDK:** Android SDK (API Level 30+)
